## express
### インストール
* `npm install express-generator -g`

### アプリ作成
* `express app`
* 依存ファイル取得：`cd app & npm install`
  * Vagrantの共有フォルダだとシンボリックリンクを作成できないためエラーになる

## Node.js
### Windows
* nodistをインストール
* `nodist + 5`
* `nodist global 5`
* `node -v`
* npmが入らなかった場合: `nodist npm global match`

### Linux
* nvmをインストール
* `nvm install 5`
